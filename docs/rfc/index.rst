Evolutions

The evolutions listed below are the planned evolutions. Each page describe how we intend to make the evolution work. If you want to speed the development of an evolution, you can:

- `contact us to finance it <mailto:contact(at)geoportal.xyz>`__
- contribute through one of our repositories on github depending on what it impacts:

    #. `geo-front3 <https://github.com/ioda-net/geo-front3>`__
    #. `geo-api3 <https://github.com/ioda-net/geo-api3>`__
    #. `geo-infra <https://github.com/ioda-net/geo-infra>`__
